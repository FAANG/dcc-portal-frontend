Frequent Asked Questions:

How do I get involved with FAANG project?

 Please first register at the FAANG website and start from there.

What is this data portal for?

 We are the FAANG Data Coordination Centre (DCC) who accepts the data submission from the FAANG community. This data portal provides an easy-to-use interface to access those submitted data

How to submit data to FAANG data portal?

 It is a not-so-easy-but-managable process and we have developed a dedicated wiki-like web page to guide you through the submission.

How to access FAANG data?

 We provide two ways of accessing the data: one is through this data portal, the other is through APIs to access programmatically. More details can be found here.

Is there a way to access data programmatically?

 Certainly. We provide APIs to access data programmatically. The details can be found here.

I still have questions which are not listed here, what shall I do?

 Please contact FAANG DCC and we welcome any type of questions.
